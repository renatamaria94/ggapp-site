# Site institucional — GGAPP (estático) — Tema claro

## Como publicar (Netlify)
- Arraste a pasta do site (ou o zip extraído) para o Netlify **ou**
- Suba para um repositório no GitHub e conecte no Netlify.

Arquivos principais:
- `index.html`
- `assets/logo.png`

## GitHub Pages
1. Suba o conteúdo para um repositório.
2. Settings → Pages → Deploy from a branch (main / root).
3. Use a URL gerada.
